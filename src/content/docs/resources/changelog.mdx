## 2026-05-26 — initial public documentation

- 🆕 **Documentation:** `docs.aggregator.gg` launched. OpenAPI 3.1 source of truth at [`/aggregator-gg/aggregator/blob/main/docs/api/openapi.yaml`](https://github.com/aggregator-gg/aggregator/blob/main/docs/api/openapi.yaml).
- 🆕 **Versioning policy:** Hybrid path + `Aggregator-Version` header model (see [Versioning](/resources/versioning/)). 180-day minimum deprecation window.
- 🆕 **`Idempotency-Key` header** documented as required on `POST /v1/sessions` and other financial endpoints.
